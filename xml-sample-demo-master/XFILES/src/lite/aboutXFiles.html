Welcome to the XFiles, an XML based Content Management system

Key features of the XFiles Application

  • Leverages the full power of the Oracle XML DB Repsository to manage XML and non-xml content
  • All HTML pages are generated using XSL to transform XML
  • Combines the power of AJAX and Database native web services to access Oracle XML DB Features
